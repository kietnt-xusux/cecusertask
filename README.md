# Base project

Laravel + React + Tailwindcss + Vite + Zustand

## Installing

You can install this project with composer and yarn

``` bash
$ composer install
$ yarn install
$ cp .env.example .env
$ php artisan key:generate
$ php artisan migrate
$ php artisan db:seed
$ php artisan storage:link
```

## Env
```
APP_URL
DB_DATABASE
DB_USERNAME
DB_PASSWORD
```

## Build

``` bash
$ yarn build
```

## Develop
``` bash
$ yarn dev
```
