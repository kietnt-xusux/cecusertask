# Base project

Laravel + React + Redux + Tailwindcss + Snowpack

## Installing

You can install this project with composer and yarn

``` bash
$ composer install
$ yarn install
$ cp .env.example .env
$ php artisan key:generate
$ php artisan migrate
$ php artisan db:seed
$ php artisan storage:link
```

## Env
``` bash
APP_URL
DB_DATABASE
DB_USERNAME
DB_PASSWORD
```

## Build

``` bash
$ yarn build
```

## Develop
``` bash
$ yarn dev
```
